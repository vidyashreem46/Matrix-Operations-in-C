# Matrix Operations using C

This project implements basic matrix operations using C programming language.
It demonstrates the use of functions and 2D arrays for modular and efficient code.

## Features
- Matrix Addition
- Matrix Multiplication
- Matrix Transpose
- Menu-driven program
- Uses functions for better modularity

## Technologies Used
- C Programming
- 2D Arrays
- Functions

## How to Run
1. Clone the repository or copy the source code.
2. Compile the program using GCC:
   gcc matrix.c -o matrix
3. Run the executable:
   ./matrix
